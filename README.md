# EcoCampo

**Projeto Agrinho 2026 - Agro Forte, Futuro Sustentável**

O EcoCampo apresenta uma fazenda sustentável que une **produção agrícola** e **preservação ambiental**.  
O site contém:

- Informações sobre produção inteligente
- Preservação do meio ambiente
- Simulador de sustentabilidade
- Quiz interativo
